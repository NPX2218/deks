## [0.1.0] - 2026-XX-XX

### Added
- Window-level workspace management via Accessibility API
- Browser tab group awareness for Chromium-based browsers
- Instant hotkey switching (⌃1–⌃9, configurable)
- Menu bar widget with workspace dropdown
- Quick switcher overlay (⌥Tab)
- Idle optimization using SIGSTOP/SIGCONT
- Named and colored workspaces
- Workspace-specific wallpapers
- macOS Focus mode integration
- Launch on login via SMAppService
- Floating windows (persist across all workspaces)
- Native HUD overlay on workspace switch
- Auto-assign new windows to active workspace
- Drag-and-drop window assignment in config panel
- JSON-based configuration in ~/Library/Application Support/Deks/
